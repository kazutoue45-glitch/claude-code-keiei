## ① キャンペーンバッジ（プレミアムゴールド・3形状）

LP 全体で最も重要な訴求「新規入塾生限定／初月授業料1ヶ月無料」を視覚的に最大化するための装飾バッジ。
ゴールドグラデーション＋二重ボーダー＋sparkle 装飾でプレミアム感を出す。**通常CTA・他セクションの強調にゴールドを流用しないこと**（希少性担保）。

### 配色

- ベース：クラシックゴールド `#D4AF37`
- ハイライト：ライトゴールド `#FFE066`
- シャドウ：ダークゴールド `#8B6914`
- 文字色：ディープバーガンディ `#7B1F1F`
- グラデーション方向：上から下に `#FFE066 → #D4AF37 → #B8860B`

### 必須装飾

- 内側に二重ボーダー（細い金線×2本、間に1〜2pxの余白）
- 周囲に sparkle ／星／輝きエフェクトを3〜5箇所（白＋ライトゴールド2色）
- 四隅または上下に小さなオーナメント（葉飾り／ダイヤ／月桂冠／クラウン）
- ドロップシャドウ（ディープバイオレット `#1A0F2E` 透過30% / blur 16px / y軸 8px）

### 1-A. `badge-medal.svg` — 円形メダル型（FV右上配置用）

- **サイズ**：200×200（SVG／拡大対応）
- **形状**：円形メダル
- **テキスト**：上段「新規入塾生限定」（14px相当・太字） / 下段「初月授業料 1ヶ月無料」（28〜32px相当・極太・「1ヶ月」「無料」を1.1〜1.2倍強調）
- **追加演出**：外周に薄い光彩（glow ／ライトゴールド透過40% / blur 24px）
- **配置先**：`§1 FV右上`（Astro 側で `top:24px; right:24px` に配置済）
- **Codex プロンプト例**：
  > 円形のプレミアムゴールドメダルバッジ。ゴールドグラデーション（上FFE066→中D4AF37→下B8860B）、二重金線ボーダー、四隅に sparkle 装飾、外周に薄い光彩。日本語テキスト「新規入塾生限定」上段と「初月授業料 1ヶ月無料」下段。文字色はディープバーガンディ#7B1F1F。SVG出力。背景透明。

### 1-B. `badge-ribbon.svg` — ワイドリボン型（料金表直下配置用）

- **サイズ**：800×140（横長）
- **形状**：左右に折り返し（テール）が出たリボン
- **テール色**：ダークゴールド `#8B6914`
- **テキスト**：上段「新規入塾生限定」（13px相当・letterspacing大） / メイン「初月授業料 1ヶ月無料」（28px相当・極太） / 下段（注釈は Astro 側で別途出すため画像に含めなくて可）
- **配置先**：`§8 料金表直下`（最も価格不安解消＋お得感を演出する位置）
- **Codex プロンプト例**：
  > 横長ワイドリボン型ゴールドバッジ。リボンの左右に折り返しテール（ダークゴールド#8B6914）あり。中央パネルはゴールドグラデーション、二重ボーダー、上下に月桂冠オーナメント、sparkle 4箇所。日本語テキスト「新規入塾生限定」上段、「初月授業料 1ヶ月無料」メイン。文字色#7B1F1F。SVG出力。背景透明。

### 1-C. `badge-mini.svg` — 小型バッジ型（アクセス内配置用）

- **サイズ**：120×120
- **形状**：円形メダル型の縮小版（メダルと同じデザイン言語、装飾は控えめに）
- **テキスト**：上段「初月」 / 下段「1ヶ月無料」（極太）

## ② §4 選ばれる3つの理由 ピクトグラム

線画＋メインパープル `#4C1D95` ＋オレンジ `#E87A2E` の2色構成で統一。**スタイル統一が最重要**（3点が同じシリーズに見えること）。

### 2-A. `reason-icon-line.svg` — 理由①「公式LINEで塾長と直接つながる」

- **サイズ**：240×240
- **モチーフ**：吹き出し（LINEチャット風）+ 塾長（先生）のシンプルなアイコン
- **Codex プロンプト例**：
  > 線画スタイルのアイコン。チャットアプリの吹き出しと、教師（眼鏡をかけた優しい雰囲気の男性）のシンプルなアバターを組み合わせ。線色はパープル#4C1D95、アクセントにオレンジ#E87A2E。背景透明。SVG出力。

### 2-B. `reason-icon-fork.svg` — 理由②「3つの学び方から選べる」

- **サイズ**：240×240
- **モチーフ**：1つの起点から3方向に分岐する図解。先端に「家（リアル校舎）」「PC（オンライン）」「訪問バッグ（家庭教師）」のアイコン
- **Codex プロンプト例**：
  > 線画スタイルのアイコン。1つの中心点から3方向に矢印が分岐し、それぞれの先に「校舎ビル」「PCモニター」「訪問バッグ」のシンプルアイコンが配置。線色はパープル#4C1D95、矢印のみオレンジ#E87A2E。背景透明。SVG出力。

### 2-C. `reason-icon-team.svg` — 理由③「8名の講師チーム」

- **サイズ**：240×240
- **モチーフ**：8人のシルエットが円形に並ぶ
- **Codex プロンプト例**：
  > 線画スタイルのアイコン。8人の人物シルエット（教師の雰囲気・性別年齢ばらけて）が円形に並ぶ。線色はパープル#4C1D95、中央の小さなアクセント（星や輪）のみオレンジ#E87A2E。背景透明。SVG出力。

## ⑥ 画像未着時のフォールバック挙動

各 `<img>` タグに `onerror` ハンドラを仕込んであり、画像がロードできなければ CSS プレースホルダー（破線枠／ゴールドダミー）が代替表示される。
ただし配置位置・サイズは確保されているため、Codex 生成画像が届くまで構成自体は崩れない。
