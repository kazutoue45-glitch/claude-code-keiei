# Changelog

Todas as mudanças notáveis neste projeto serão documentadas aqui.

Formato baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/).

## [1.0.0] - 2026-03-27

### Adicionado
- Skill principal (`SKILL.md`) com 11 fases de execução
- Fase 0: Setup com 5 perguntas interativas (modo, objetivo, nicho, velocidade, hospedagem)
- Fase 1: Briefing com dois caminhos (documento pronto ou questionário de 10 perguntas)
- Fase 2: SEO Strategy via `/seo-plan`
- Fase 3: Copy com 3 caminhos (pronta, do zero, parcial) via `/copywriting`
- Fase 4: Design com inventário visual, Stitch opcional, tokens MD3, fontes por nicho
- Fase 5: Build de HTML único com 12 seções e 8 funcionalidades JS inline
- Fase 6: SEO Técnico via `/seo-geo` + `/seo-schema` + `/seo-page` em paralelo
- Fase 7: Páginas auxiliares (download com toasts + thank-you com tracking)
- Fase 8: QA em 3 camadas (checklist 42 checks + `/design-review` + Playwright 18 testes)
- Fase 9: Deploy SFTP automático ou ZIP com guia personalizado
- Fase 10: Entrega com `ENTREGA-TECNICA.md` + `PROXIMOS-PASSOS.md`
- Suporte a 7 nichos: jurídico, saúde, SaaS, educação, imobiliário, estética, genérico
- Suporte a 5 objetivos: lead capture, venda, agendamento, download, evento
- Dois modos de velocidade: rápido (3 checkpoints) e detalhado (7 checkpoints)
- Templates de entrega com placeholders
- Documentação de referência para cada fase
- Exemplo completo: case jurídico (restituição de consórcio)
- README com instruções de instalação e uso
- CONTRIBUTING.md com guia para contribuidores
