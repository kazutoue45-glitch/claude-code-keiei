Q

公式LINEに登録したら、しつこく勧誘されませんか？

A

ご安心ください。義塾の公式LINEは、塾長との相談窓口としてご利用いただくものです。ご家庭の状況をうかがった上でご案内するため、無理な勧誘や強引な営業は一切いたしません。ご相談だけで終わっていただいて構いません。

Q

無料体験はできますか？

A

はい、公式LINE登録後、塾長との個別相談を経て無料体験授業をご案内します（リアル校舎・オンライン・家庭教師いずれも対応）。

Q

途中で退塾する場合の手続きは？

A

月末締めでのご退塾手続きが基本です。違約金等は発生しません。

Q

部活や習い事と両立できますか？

A

はい、リアル校舎は17:00〜21:40の中から時間帯を選択でき、家庭教師は曜日・時間をご家庭の都合に合わせて調整します。オンラインなら移動時間ゼロで両立しやすくなります。

Q

途中で指導形態（リアル↔オンライン↔家庭教師）を変更できますか？

A

状況に応じて柔軟にご相談いただけます。塾長との個別面談で最適な形をご提案します。

Q

受験対策はどの学年から始めればよいですか？

A

中学受験は小3〜小5、高校受験は中1〜中2、大学受験は高1からのスタートが理想です。学年や状況に応じて、塾長が最適なプランをご提案します。
