FACILITY

2校舎に共通する設備・サポート

自習室

平日17時から開放。塾生は自由に使えます。家では集中できないお子さまに最適。

土曜質問クラス

毎週土曜19:00〜22:00、5教科対応の無料質問クラス。塾生は何度来ても無料。

進路相談室

個別面談用のスペースを完備。塾長が直接、お子さまと保護者様のご相談にお答えします。

遅い時間帯対応

20:00〜21:50の遅い時間にも対応。クラブ活動と勉強を両立できる時間割です。

教室を見学にいらっしゃいませんか？

実際の教室の雰囲気を、お子さまと一緒にご確認ください。見学・個別相談はもちろん無料です。
