成績で、
生徒を選ばない。

たとえ時代がどのようにぶれようが、子供中心の視点から子供達が本当に望んでいることに応え、そして親の子供に対する熱い思いに共感することが、塾本来の役割だと、私は思っています。

学校の先生より、塾の私たちは「本音で接する立場」にいます。教育現場の最前線で、子供達の小さな変化に気づき、保護者の皆様と並走する。それがフォレストの仕事です。

FOREST STRENGTHS

ホンモノの個別指導は
「つきっきり」でなければならない。

フォレストの個別指導は、ただ机を並べるだけの時間ではありません。一問ごとに、生徒の理解度・スピード・表情まで見つめながら進める。それが、私たちが25年続けてきた指導の形です。

01

つきっきりの個別指導

1対2・1対3でも、わからないを見逃さない。問題を解くスピード・理解度・やる気の3つを毎回見ています。

02

成績で生徒を選ばない

「できる子」だけ集めても本当の指導力は身につきません。一人ひとりの今からの伸びに本気で向き合います。

03

毎週土曜・無料質問クラス

19:00〜22:00、5教科対応の自由質問クラス。塾生は何度来ても無料。「わからない」を翌週に持ち越しません。

04

中学受験 合格率100%

2001年の開校以来、中学受験は合格率100%の実績。常翔学園・信愛女学院など地元中堅私学に対応。

05

平日17時から自習室開放

「集中して勉強できる場所がある」だけで、家では出せない力が出ます。塾生は自由に使えます。

06

クラブ活動と両立できる時間割

20:00〜21:50の遅い時間帯にも対応。部活で疲れた日も、無理なく通い続けられる仕組みです。
