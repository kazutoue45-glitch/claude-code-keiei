フォレストの強み

つきっきり個別指導の中身

ホンモノの個別指導は「つきっきり」でなければならない。
2001年の開校から、私たちが一度も譲らずに続けてきた指導の形を6つの軸でご紹介します。

01 つきっきり個別指導

1対2・1対3でも、一人ひとりの「わからない」を見逃さない。

「個別指導」と一口に言っても、生徒が黙々と問題を解いている横で講師が座っているだけ……そんな塾も少なくありません。フォレストは違います。

問題を解くスピード、理解度、表情、その日のやる気。一回ごとに複数の角度から見つめながら、講師が常に「今、何を伝えるべきか」を判断します。これが、私たちの言う「つきっきり」の意味です。

ホンモノの個別指導は「つきっきり」でなければならない。問題を解くスピード、理解度、やる気など、様々な角度から生徒を見つめてこそ、本当の指導ができる。
02 50分・100分の集中システム

集中が切れない、独自の100分システム。

100分コースは、50分授業 → 10分休憩 → 50分授業の構成。「長時間座っていられない」「集中が続かない」というお子さまでも、メリハリをつけて学習できる仕組みです。

  • 50分単位で講師が指導内容を切り替え、飽きさせない
  • 10分休憩で頭をリセットし、後半の理解度を最大化
  • 1対2・1対3指導で、一人にかける時間と密度を両立
03 毎週土曜・無料質問クラス

土曜日19時〜22時、5教科対応の自由質問クラス。

塾生なら何度来ても無料。複数の講師が常駐し、5教科すべてに対応します。「平日に消化しきれなかった疑問」「学校の宿題でつまづいたところ」、何でも持ってきてください。

「わからない」を翌週に持ち越さない。これが、フォレストが一番大切にしている習慣です。

04 中学受験 合格率100%

2001年の開校以来、中学受験は合格率100%。

常翔学園中学・信愛女学院中学など、地元中堅私学への合格実績を毎年積み重ねています。「大手塾の一斉授業についていけなかった」というお子さまでも、フォレストの個別指導で合格を勝ち取ってきました。

「成績で生徒を選ばない」という方針は、中学受験でも変わりません。今の偏差値ではなく、これからの伸び代に本気で向き合います。

05 平日17時から自習室開放

「集中して勉強できる場所」が、家以外にある。

平日17時から、塾生は自由に自習室を使えます。家ではついスマホを触ってしまう、兄弟がいて集中できない――そんなお子さまでも、自習室なら勉強モードに切り替えられます。

自習中もすぐ近くに講師がいるので、わからないことがあれば気軽に質問できます。

06 進路相談・入試情報サポート

地元の高校情報を、独自に集約してお届け。

大阪府公立高校・私立高校の入試情報を、各校から独自にヒアリングして集約しています。模試の偏差値だけでは見えない「学校の雰囲気」「先生のタイプ」「クラブ活動の温度感」まで含めて、お子さまに合う進路を一緒に考えます。

個別進路面談は無料。保護者様もぜひ一緒にご相談ください。

話を聞きにきませんか？

無料の個別相談で、お子さまの今の状況をお聞かせください。塾長が直接お話しします。
